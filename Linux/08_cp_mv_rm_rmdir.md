# cp, mv, rm , rmdir

1. cp: copy file

- cp __file_gốc __file_cp
- cp __dir_gốc __dir_cp
- cp -r __dir_gốc __dir_cp

2. mv: move file

- mv __file_gốc __file_dir
- mv __dir_gốc __dir_dir
- mv -r __dir_gốc __dir_dir

3. rm: xóa file

- rm __filename
- rm -rf __dir

4. rmdir: xóa thư mục

- rmdir __dir
